# CONNECTA
Connecta é um sistema para trabalho de curso, onde é para ter a conexão de professor alunos, com base em posts, onde o professor posta e o aluno vê, logo só pode visualizar, reagir e chamar o professor para perguntar sobre, ele é mais para conexão em sala de aula mesmo, bem simples e o sistema em si foi desenvolvido somente para demonstração em sala de aula com pouca função de fato.
